CSharpEssentials.LoggerHelper

Route Serilog sinks by log level — zero boilerplate, native ILogger<T> support.

Install the NuGet, add 5 lines of JSON, and every ILogger<T> in your app automatically routes logs to Console, File, Email, Telegram, PostgreSQL, SQL Server, Elasticsearch, Seq, and Hangfire Console — each sink receiving only the levels you want.

dotnet add package CSharpEssentials.LoggerHelper
dotnet add package CSharpEssentials.LoggerHelper.Sink.Console
dotnet add package CSharpEssentials.LoggerHelper.Sink.File
